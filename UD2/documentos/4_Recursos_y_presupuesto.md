## **4. Recursos y presupuesto**

### ✅ **Recursos técnicos**

- Ordenador personal para desarrollo.  
- Máquinas virtuales para entornos de prueba.  
- Servidor local o VPS.  
- Software open source (sin coste).  
- Docker para la contenedorización.  
- GitHub para control de versiones.

---

### ✅ **Presupuesto estimado**

| Concepto | Coste aproximado |
|----------|-------------------|
| VPS para pruebas | 10–20€/mes |
| Dominio opcional | 10€/año |
| Herramientas de desarrollo | 0€ |
| Coste total aproximado | 10–20€/mes |

El coste real es mínimo porque el proyecto se basa en herramientas gratuitas de código abierto.
