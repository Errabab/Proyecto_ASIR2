## **3. Evaluación del proyecto**

La evaluación del proyecto se ha realizado comprobando que se cumplen los objetivos planteados en la UD2 y que el sistema funciona de forma estable dentro del entorno AWS proporcionado por el centro.

### ✅ **Criterios de evaluación cumplidos**

- **Funcionalidad completa:**  
  El sistema realiza auditorías web generales y auditorías específicas de WordPress.

- **Automatización:**  
  Los escaneos pueden programarse con cron para ejecutarse automáticamente.

- **Panel web operativo:**  
  El panel muestra resultados, históricos y estado general de la seguridad.

- **Base de datos funcional:**  
  MariaDB almacena correctamente los informes y auditorías realizadas.

- **Alertas activas:**  
  El bot de Telegram envía avisos cuando se detecta una vulnerabilidad importante.

- **Estabilidad del sistema:**  
  Todos los contenedores Docker funcionan sin errores críticos.

- **Cumplimiento del tiempo:**  
  El proyecto se ha finalizado dentro del plazo (15 de diciembre), siguiendo la planificación ajustada.

### ✅ **Consideraciones finales**
El proyecto demuestra que es posible construir una solución de auditoría web profesional utilizando únicamente herramientas gratuitas, un servidor AWS educativo y tecnologías open source.

Además, la plataforma tiene potencial real para convertirse en un servicio para PYMES.
