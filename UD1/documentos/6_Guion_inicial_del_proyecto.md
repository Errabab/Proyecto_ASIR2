## **6. Guion inicial del proyecto**

1. Desarrollo de scripts de análisis web (Nmap, Nikto, SSL Scan, headers…).  
2. Integración de la API de OWASP ZAP para auditorías automatizadas.  
3. Implementación del módulo WordPress con WPScan.  
4. Sistema de almacenamiento de resultados.  
5. Creación del panel web para visualización.  
6. Sistema de alertas por email y Telegram.  
7. Automatización mediante cron o servicios internos.  
8. Pruebas reales con webs controladas.  
9. Documentación final del proyecto.
