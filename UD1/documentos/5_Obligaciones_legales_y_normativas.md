## **5. Obligaciones legales y normativas**

El proyecto se ajusta a varias normativas y buenas prácticas:

- **RGPD**: el sistema no trata datos personales; solo analiza configuraciones públicas.  
- **LSSI-CE**: se requiere autorización del propietario del dominio para escanear.  
- **Ética en ciberseguridad**: toda auditoría debe realizarse de forma responsable.  
- **OWASP Top 10**: guía referencia para identificar los riesgos de seguridad web.  
- **ISO/IEC 27001**: se toma como referencia para buenas prácticas de seguridad.  

El proyecto sigue un enfoque totalmente seguro y responsable.
